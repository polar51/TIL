#### 1레벨 추억 점수

- 문제를 너무 어렵게 생각하긴함
- 마찬가지로 여유가 없었다.

```py
def solution(name, yearning, photo):
    # 1. 이름과 그리움 점수를 딕셔너리로 매핑 (zip 사용)
    # { '이름': 점수 } 형태로 빠르게 생성
    person_dict = dict(zip(name, yearning))

    # 2. 각 사진(photo)의 점수를 계산하고 최종 리스트(answer)에 저장 (리스트 컴프리헨션 사용)
    answer = [
        # 각 사진(picture)에 대해 내부 합계를 계산
        sum(
            # 각 이름(person)에 해당하는 점수를 가져옴 (없으면 0)
            person_dict.get(person, 0)

            # 사진 속 모든 사람을 순회
            for person in picture
        )
        # 모든 사진을 순회
        for picture in photo
    ]

    return answer
```

- 내가 짠 코드도 동작은 하지만 위 형태가 더 파이토닉스럽고 간결하다 dict(zip(a,b))는 외워두고 아래에 get 함수도 외워두자
- 특히 컴프리헨션은 이미 내가 알고있지만 유연하게 생각하지못해서 사용하지 못하는 경우가 많은 반복문을 쓰기전에 한번 생각해보자
- get 함수는 딕셔너리의 값을 안전하게 가져오는 함수이다. 이름으로 사람의 그리움 점수를 디셔너리에서 받아오는데 이때 키 값이 없는 경우에 대한 기본 값을 get(a, b)로 b로 해두기때문에 a가 없으면 b를 반환한다.
