#### 1레벨 카드 뭉치

- 약간 어거지로 풀긴 풀었다. 솔직히 자바스크립트에서 자주 사용되는 부분이라 편하게 풀긴했지만 인덱스 0번이 없는데 계속 0번을 조회하는 부분에서 에러가 났다. `len(배열) > 0`으로 안전하게 처리해주었다면 더 좋았을텐데

```py
def solution(cards1, cards2, goal):
    for word in goal:

        # cards1이 비어있지 않고(True), 첫 번째 요소가 일치하는지 확인
        if cards1 and cards1[0] == word:
            cards1.pop(0)

        # cards2가 비어있지 않고(True), 첫 번째 요소가 일치하는지 확인
        elif cards2 and cards2[0] == word:
            cards2.pop(0)

        # 둘 다 해당되지 않으면 (만들 수 없는 경우)
        else:
            return 'No'

    return 'Yes'
```

- 이런 식으로 푸는게 괜찮은 풀이법인데 cards1으로 하면 배열안에 값이 있는 경우에 True 값이 없으면 False를 반환한다. 또 remove 대신 pop이 더 추천되는데 그 이유는 문제에서 **순서대로 꺼낸다** 라는 부분인데 remove는 배열안의 해당 값을 삭제하지만 pop은 pop(0)을 통해 0번 인덱스를 삭제한다는 안정성이 있다. remove와 pop의 차이를 알고 쓰고 배열의 인덱스를 생각하자
