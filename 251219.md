#### 1레벨 둘만의 암호

```py
import string

def solution(s, skip, index):
    answer = ''
    skip_set = set(skip)
    alphabet = [char for char in string.ascii_lowercase if char not in skip_set]

    for a in s:
        i = alphabet.index(a)

        idx = (i + index) % len(alphabet)

        answer += alphabet[idx]

    return answer
```

- 처음으로 시간 초과해서 실패함;;
- import string 모듈로 string.ascii_lowercase 라고 선언해주면 바로 소문자 전체를 가져옴.. 이걸 하나씩 손으로 치고있었다니...
- skip 이라는 배열을 먼저 set 으로 만들고 컴프리헨션으로 아래와 같은 로직을 컴프리헨션으로 가져온다.

```py
alphabet = []
for char in string.ascii_lowercase:
    if char not in skip_set:
        alphabet.append(char)
```

- %연산자는 몫으로 나눈 나머지라는 뜻으로 순환해야하는 부분에 쓰기 좋다.
  ```py
  (5 + 2) % 8 = 7
  (8 + 8) % 16 = 0
  (2 + 8) % 5 = 0
  (2 + 8) % 4 = 2
  ```
